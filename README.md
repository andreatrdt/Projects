# Projects

A collection of financial engineering and machine-learning projects.

## Hawkes Optimal Control

The `HawkesOptimalControl` project now has three layers:

1. Hawkes modelling of aggressive order-flow.
2. HJB-based optimal market-making control.
3. ML-based short-horizon predictive intervals for the next mid-price.

The ML layer is in `HawkesOptimalControl/code/ml_interval_forecaster.py` and is demonstrated in `HawkesOptimalControl/code/main.ipynb`.
